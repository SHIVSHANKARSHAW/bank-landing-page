## Installation
To install the project, clone the repository and install the dependencies:
```bash
git clone https://github.com/SHIVSHANKARSHAW/project_hoobank-main.git
cd project_hoobank-main
npm install
```

## Usage
To start the development server, run:
```bash
npm run dev
```
